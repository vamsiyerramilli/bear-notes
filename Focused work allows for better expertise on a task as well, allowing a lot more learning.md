# Focused work allows for better expertise on a task as well, allowing a lot more learning
Brain has to “connect and disconnect” a lot less when it works on a single task / related set of tasks in depth instead of switching between a set of random tasks. Allows for a state of flow to develop and it becomes almost effortless to work through the problems / tasks at hand
